### MINIMUMS DIALOG BOX

Figure 5-46 Minimums Dialog Box

The MINIMUMS dialog box allows the pilots to select between the Radio Altimeter (RA) or Barometric (BARO) selections for the approach altitude reference shown on the altimeter display. Setting the values on one dialog box set the values for the cross-side also.

SELECTIONS:

OFF The approach references do not show on the altimeter and no aural indication is given.

RA Allows the pilots to set the RA value and places the information on the altimeter.

BARO Allows the pilots to set the BARO value and places the information on the altimeter.

Embraer Legacy 450/500 and Praetor 500/600 MENUS AND DISPLAYS Pro Line Fusion® Avionics Tuning

• Select the gray ADF button to access the ADF Control submenu.

• Select the transfer (up/down arrow) button to transfer the active and standby ADF frequencies.

• Select the standby frequency to tune it.

• Use the cursor to change the ADF mode, BFO setting, and to test the ADF radio.

HF

The HF section of the Tuning format provides simple controls to modify HF settings.

• Select the gray HF button to access the HF Control submenu.

• Use the cursor to select between Lower Voice (LV) single sideband, Upper Voice (UV) single sideband, and Amplitude Modulation (AM).

• Use the cursor to select the squelch level, HF mode, and power level.

• Select the frequency readout to tune the HF frequency or channel.

• HF operation is selectable for simplex, duplex, emergency, and maritime modes.

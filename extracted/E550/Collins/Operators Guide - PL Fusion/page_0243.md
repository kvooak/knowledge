Embraer Legacy 450/500 and Praetor 500/600 MENUS AND DISPLAYS Pro Line Fusion® Avionics Tuning – XPDR/TCAS Control

ID Set the ID code by positioning the cursor over the ID code window and using the DSK. The outer knob moves the focus indicator along the text string, and the inner knob changes the character inside the focus indicator. The ﬂight ID code may also be entered with the MKP.

ALT Tag ALT Tag is a multi-state toggle to select the altitude tag format for TCAS trafﬁc to Relative (REL) or Absolute (ABS).

XPDR SEL XPDR SEL is a multi-state toggle to select the XPDR1 or XPDR2 transponder.

◊TRAFFIC ◊TRAFFIC is a multi-state toggle to enable or disable the display of non-threat (◊) TCAS trafﬁc.

Mode Mode is a multi-state toggle to select the XPDR/TCAS mode. The following selections are provided:

• AUTO – Automatic mode

• STBY – Standby

• TA/RA – Trafﬁc Advisory (TA)/ Resolution Advisory (RA)

• TA ONLY – Trafﬁc Advisory (TA) Only

• ALT ON – (Transponder only, mode C)

• ALT OFF – (Transponder only, mode A).

ALT Limit ALT Limit is a multi-state toggle to select the Other Trafﬁc altitude window. The operator can choose from the following selections:

• NORM

• ABOVE

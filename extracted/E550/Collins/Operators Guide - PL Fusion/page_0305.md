Embraer Legacy 450/500 and Praetor 500/600 MENUS AND DISPLAYS Pro Line Fusion® Avionics Map – Vertical Situation Display

Altitude Tape The VSD displays an altitude tape on the Y (vertical) axis. The altitude tape displays the BARO-corrected altitude in feet. If the map range is 300 NM or greater, the altitude tape is ﬁxed at a maximum of 50 000 feet. If the map range is 200 NM or less, the altitude tape displays an optimum range. The optimum range varies the altitude increment and total shown range based on the map range/distance tape:

Lateral Reference Frame

The lateral reference frame of the VSD can be ﬂight plan or track. The lateral reference frame is ﬂight plan except for the following conditions:

• Active FD mode is not LNAV

• GPS source failure

• Difference between current track and desired track is greater than 90 degrees

• Active FMS leg guidance is invalid

• No ﬂight plan

• Discontinuity

• Sequence inhibit

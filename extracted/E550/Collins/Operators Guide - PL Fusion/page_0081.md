Embraer Legacy 450/500 and Praetor 500/600 CONTROLS AND INDICATORS Pro Line Fusion® Avionics Flight Control Panel

BANK Half Bank Selection Button

The BANK button selects and deselects a reduced bank angle limit. It will limit toggle between ±15 degrees and ±30 degrees.

HDG Heading Mode Select Button

The HDG button selects and deselects heading select mode. When in this mode, the AFCS generates commands to capture and maintain the selected heading shown on the PFD. The selected heading is set by the HDG knob.

HDG Heading Selection Knob

The HDG knob selects the heading, it is displayed on the HSI on the PFD. The HDG button synchronizes the heading bug with the current airplane heading.

PUSH SYNC Push Sync Button

Push the PUSH SYNC button to align the HDG reference bug to the current heading, which places the reference bug directly beneath the heading readout.

APPR Approach Mode Select Button

The APPR button selects and deselects approach mode. When approach mode is selected, lateral commands are generated to capture and track management system. After the lateral mode is captured, vertical commands are generated to capture and track the vertical path from the Glideslope receiver or ﬂight management system.

AP Autopilot Control Switch

The AP button engages and disengages the autopilot. If not engaged, the FD will engage automatically along with the autopilot.

CONTROLS AND INDICATORS Embraer Legacy 450/500 and Praetor 500/600 Display Control Panel Pro Line Fusion® Avionics

### DISPLAY CONTROL PANEL

Two Display Control Panel (DCP) are mounted in the glare shield. The DCP allow the crew to show or remove the SVS, TCAS, and WX, change the HSI format, select the primary NAV source, and adjust the barometric pressure.

CALLOUTS:

CVS Combined Vision System Button

The Combined Vision System (CVS) button places the Synthetic Vision System (SVS) image on the PFD in place of the traditional brown and blue background. The second push turns the SVS off. For more information, refer to the SVS section of the Menus and Displays chapter of this Operator’s Guide. The SVS is only available on the full and half PFD displays.

FPV CAGE Flight Path Vector Cage Button

The FPV Cage button centers the FPV on the PFD when selected. The second push returns the FPV to normal indications. The selection is only enabled when SVS is active. For more information refer to the FPV section of the Menus and Displays chapter of this Operator’s Guide.

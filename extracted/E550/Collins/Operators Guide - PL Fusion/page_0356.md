MENUS AND DISPLAYS Embraer Legacy 450/500 and Praetor 500/600 XM GWX – Meteorological Report Pro Line Fusion® Avionics

Figure 5-160 XM GWX – Meteorological Report Menu

The Meteorological Report menu provides access to textual National Meteorological reports. If no SIGMET or AIRMET reports have been received by the system, the respective menu selection is disabled.

SELECTIONS:

SIGMET SIGMET is a submenu access button to the SIGMET submenu. The SIGMET submenu is a dialog box to access textual SIGMET reports.

AIRMET AIRMET is a submenu access button to the AIRMET submenu. The AIRMET submenu is a dialog box to access textual AIRMET reports.

MENUS AND DISPLAYS Embraer Legacy 450/500 and Praetor 500/600 Chart – Chart Task Menu Pro Line Fusion® Avionics

The Chart Task menu provides controls to format the chart currently shown in the DU Chart. The Chart Task menu is accessed by selecting the chart background.

SELECTIONS:

Available Charts Available Charts is a submenu select to show the Available Charts dialog box. The Available Charts dialog box provides controls to change the current chart to another chart of the same type. If no more than one chart of the current type for the current airport is available, the Available Charts submenu select is disabled.

Split Split is a single stage toggle to enable the graphical approach chart to show in the split format.

Orient Orient is a button to change the orientation of the shown chart between portrait and landscape.

Dim Dim is a single stage toggle to select the night background for the current chart.

Center Aircraft Center Aircraft is a button to center the shown chart on the current aircraft position within the window. If the chart is not geo-referenced, or the aircraft is not on the body of the chart, the Center Aircraft selection is disabled.

Center Chart Center Chart is a button to center the chart in the current window, maintaining the current zoom level.

Fit Fit is a button to size the chart to ﬁt the current window.

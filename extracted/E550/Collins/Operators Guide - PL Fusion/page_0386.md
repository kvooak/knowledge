MESSAGES AND ANNUNCIATORS Embraer Legacy 450/500 and Praetor 500/600 Visual Annunciations Pro Line Fusion® Avionics

ADI: FPV Flag

#### .

A boxed red FPV ﬂag is shown to the left of the attitude indicator when ﬂight path data is missing or invalid. The ﬂag ﬂashes for 5 seconds and then changes to a steady-on state.

FPV indications are removed when the FPV ﬂag is being shown.

ADI: Glideslope Flag

#### .

A boxed red GS ﬂag is shown at the lower end of the vertical deviation scale when a Localizer (LOC) is the active NAV source and the vertical deviation input from the active NAV source is missing or invalid. The ﬂag ﬂashes for 5 seconds and then changes to a steady-on state.

The diamond vertical deviation pointer is removed when the GS ﬂag is being shown.

HSI: Heading Flag

The boxed, red heading ﬂag comes into view if both primary and battery power to the AHC is lost. The ﬂag ﬂashes for 5 seconds and then changes to a steady-on state.

All heading references are all removed when the HDG ﬂag is being shown.

MENUS AND DISPLAYS Embraer Legacy 450/500 and Praetor 500/600 Chart – Chart Type Menu Pro Line Fusion® Avionics

### CHART – CHART TYPE MENU

Figure 5-136 Chart – Chart Type Menu

The Chart Type menu provides submenu select buttons for the Any Chart submenu selection. The Chart Type menu is accessed through an Any Chart shortcut submenu button on a Chart menu (Origin, Destination, Alternate, or Other).

SELECTIONS:

Approach Approach is a submenu select button to access the approach chart submenu for the associated airport.

Airport Airport is a submenu select button to access the airport chart submenu for the associated airport.

Departure Departure is a submenu select button to access the departure chart submenu for the associated airport.

Arrival Arrival is a submenu select button to access the arrival chart submenu for the associated airport.

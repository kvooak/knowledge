Embraer Legacy 450/500 and Praetor 500/600 MENUS AND DISPLAYS Pro Line Fusion® Avionics Tuning – Edit HF Preset Frequencies

If the mode is Simplex, one frequency window shows. If the mode is Duplex, two Set Frequency windows show. If the mode is Emergency or Maritime, no Set Frequency windows show.

Deﬁne Name The Deﬁne Name window provides control to edit the shown name for the HF preset frequencies. Select the Deﬁne Name window to edit its contents.

Return to HF CTRL Select the Return to HF CTRL pushbutton to close the Edit HF Preset Frequencies window. The Edit HF Preset Frequencies window may also be closed by selecting the gray X pushbutton in the upper right corner.

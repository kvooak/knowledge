Embraer Legacy 450/500 and Praetor 500/600 MENUS AND DISPLAYS Pro Line Fusion® Avionics Synoptic – Environmental Control System Page

Figure 5-61 Synoptic – Environmental Control System Page

The Environmental Control System (ECS) Synoptic page graphically represents the ECS. The ECS Synoptic page is accessed through the Synoptic drop down menu accessed with the SYS QAK.

SELECTIONS:

Out Flow Valve The ECS Synoptic displays the position of the Out Flow Valve (OFV).

Set Cockpit (CKPT) and cabin temperatures can be set through OEM supplied controls. The controls provide indications on the ECS Synoptic page in cyan numbers, both in °C and °F under a white SET legend.

Actual Cabin and CKPT actual temperature is depicted in green numbers in both °C and °F under a white ACTUAL legend.

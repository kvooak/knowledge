MENUS AND DISPLAYS Embraer Legacy 450/500 and Praetor 500/600 ADI – Synthetic Vision Pro Line Fusion® Avionics

### ADI – SYNTHETIC VISION

Figure 5-21 ADI – SVS

The Synthetic Vision System (SVS) uses a terrain database and current position information to generate virtual landscape images for display on the ADI or HUD. The SVS can replace the normal sky/ground depiction that makes up the background of the ADI.

The SVS image can be shown on both the half or full screen ADI, or on the HUD. The SVS image is selected for display on the PFD Format dialog box on the PFD Task Menu.

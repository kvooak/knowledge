Embraer Legacy 450/500 and Praetor 500/600 MENUS AND DISPLAYS Pro Line Fusion® Avionics Tuning

Figure 5-74 Tuning (2/2)

All radios can be tuned with the on-screen menus. The Tuning format provides controls to tune the radios and change the settings. The sections on the main Tuning format pages provide simple controls for the respective radios, while the Control menus provide advanced controls.

SELECTIONS:

COM The COM section of the Tuning format provides controls to modify COM settings. COM1, COM2, and COM3 sections are provided.

• Select the gray COM button to access the COM Control submenu.

• Select the transfer (up/down arrow) button to transfer the active and standby VHF frequencies.

• Select the standby frequency to tune it.

Embraer Legacy 450/500 and Praetor 500/600 MENUS AND DISPLAYS Pro Line Fusion® Avionics Tuning – ADF Control

SELECTIONS:

Tune Tune the standby frequency by positioning the cursor over the standby frequency digits and using the DSK.

↕ ↕is a button to transfer the active and standby frequencies.

Mode Mode is a multi-state toggle to select the operating mode of the ADF. The operator can select ADF or ANT (Antenna).

BFO BFO is a multi-state toggle to enable or disable the Beat Frequency Oscillator (BFO) function.

Preset Frequencies The lower control menu lists the preset frequencies, controlled by a scroll list. Select a preset frequency to set it as the active frequency.

Edit EDIT is a button to show the Edit ADF Preset Frequency dialog box.

Test Test is an action command to initiate a test of the ADF radio.

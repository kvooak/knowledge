MENUS AND DISPLAYS Embraer Legacy 450/500 and Praetor 500/600 Tuning Pro Line Fusion® Avionics

• Use the cursor to change test, FREQ SEL, squelch, and mode settings.

NAV The NAV section of the Tuning format provides simple controls to modify NAV settings. NAV1 and NAV2 sections are provided.

• Select the gray NAV button to access the NAV Control submenu.

• Select the transfer (up/down arrow) button to transfer the active and standby NAV frequencies.

• Select the standby frequency to tune it.

• Use the cursor to change marker sensitivity, NAV tuning to auto or manual, and to test the NAV radio.

XPDR TCAS The XPDR TCAS section of the Tuning format provides simple controls to modify transponder/TCAS settings.

• Select the gray XPDR TCAS button to access the XPDR/TCAS Control submenu.

• Select the IDENT button to transmit an Identiﬁcation code signal.

• Use the cursor on the TCAS mode display to change the TCAS mode, ALT limit, ALT tag, and ATC proximate trafﬁc settings.

• Select the XPDR code display to tune the ATC code.

ADF The ADF section of the Tuning format provides simple controls to modify ADF settings.

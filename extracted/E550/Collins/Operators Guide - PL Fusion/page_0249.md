Embraer Legacy 450/500 and Praetor 500/600 MENUS AND DISPLAYS Pro Line Fusion® Avionics Tuning – Edit HF Preset Frequencies

Emissions Emissions is a multi-state toggle to select the HF emissions to Lower Voice (LV), Upper Voice (UV), or Amplitude Modulation (AM).

Preset Frequencies The lower control menu lists the preset frequencies, controlled by a scroll list. Select a preset frequency to set it as the active frequency.

Edit Edit is a pushbutton to show the Edit HF Preset Frequency dialog box.

Test Test is a pushbutton to initiate a test of the ADF radio.

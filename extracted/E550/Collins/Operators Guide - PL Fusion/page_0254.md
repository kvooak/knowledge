MENUS AND DISPLAYS Embraer Legacy 450/500 and Praetor 500/600 Tuning – Error Messages Pro Line Fusion® Avionics

Figure 5-89 Tuning – Left-Side Display Tuning Inhibited

The Left-Side Display Tuning Inhibited message is displayed when the pilot side RSP (Reversion Select Panel) TUNE knob has been placed in the L MKP ONLY position. When the knob is placed in this position, display of COM tuning is inhibited on the display units. The COM1/NAV1 radios can still be tuned, but they must be tuned via the MKP. A similar inhibit message is displayed when the copilots RSP TUNE knob is placed in the R MKP ONLY position.

DISPLAY OPERATION Embraer Legacy 450/500 and Praetor 500/600 GUI Elements Pro Line Fusion® Avionics

• A copy only ﬁeld is a special visual tool used in ICDU dialog boxes or format windows to indicate that the contents of the ﬁeld may be copied to the MKP scratchpad if blank, but the contents cannot be modiﬁed.

• A DU notebook is a set of GUI control groups organized as a notebook, with tabs as selection controls. Only one tab may be active at a time. Position the cursor over a tab on the DU notebook and select it to open the tab.

• The DU expand/collapse functionality is used to facilitate viewing data when individual items can contain numerous related sub-items. Position the cursor over the expand/collapse pushbutton and select it to activate the function.

MENUS AND DISPLAYS Embraer Legacy 450/500 and Praetor 500/600 CDTI Features Pro Line Fusion® Avionics

• Trafﬁc selection allows the crew to obtain additional data for any one target displayed on the MFW Map that has ADS-B IN data. Selected trafﬁc is distinguished from other trafﬁc symbols by the presence of a spotlight on the symbol. The spotlight is a circular light gray background behind the symbol, used to allow quick visual acquisition of selected trafﬁc on the display.

• Flight ID readout for all trafﬁc on the map, selectable as an overlay on the TRAFFIC sub-menu.

• ID, range, ground speed, and type of aircraft for target selected by the crew.

• Non-threat target range is >200 NM.

• Both airborne and aircraft on the ground are displayed.

N

NOTE

CDTI targets missing directional data (heading) are displayed as diamond shaped.

N

NOTE

For CDTI targets, the arrowhead is embedded within RA and TA symbols.

N

NOTE

Only trafﬁc with ADS-B in data may be selected using trafﬁc highlighting.

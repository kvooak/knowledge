Embraer Legacy 450/500 and Praetor 500/600 MENUS AND DISPLAYS Pro Line Fusion® Avionics Tuning – Select NAVAID

Deﬁne Name The Deﬁne Name window provides control to edit the shown name for the NAV preset frequencies. Select the Deﬁne Name window to edit its contents.

Return to NAV CTRL

Select the Return to NAV CTRL pushbutton to close the Edit NAV Preset Frequencies window. The Edit NAV Preset Frequencies window may also be closed by selecting the gray X pushbutton in the upper right corner.

### TUNING – SELECT NAVAID

Figure 5-82 Tuning – Select NAVAID

The Select NAVAID page shows when the NAV radio has been tuned by identiﬁer and more than one frequency is associated with that identiﬁer. The Select NAVAID page provides controls to select the correct frequency for the intended identiﬁer.

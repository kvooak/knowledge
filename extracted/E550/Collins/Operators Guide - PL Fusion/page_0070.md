CONTROLS AND INDICATORS Embraer Legacy 450/500 and Praetor 500/600 Audio Control Panel Pro Line Fusion® Avionics

CALLOUTS:

Microphone Select Buttons

The microphone select buttons select the destination for microphone audio and set the selection as the active transmitter. When a button is pressed to select a particular transmitter, the green LED above the button is turned on and all other select buttons are blocked. The receive audio of the active (selected) transmitter is automatically routed to the headphones and speaker (when not muted).

Audio Control Knobs

The audio control knobs provide for the selection and volume control of the associated received audio signal that is sent to the headphones and speaker (when not muted). The knobs are pushed in to turn the associated receive audio off unless the microphone select button is active for that radio. Pushing the knob in again unlatches the knob to the popped out position for selection and volume control of the desired receive audio. Audio can be received from any combination of radios.

INPH Interphone Volume Control

The INPH knob controls the volume of the interphone audio.

VOICE/BOTH/ID Navigation Re- ceiver Filter Select

The VOICE/BOTH/ID knob selects the received Voice, both the received voice and Morse code identiﬁer (BOTH), or just the received Morse code Identiﬁer (ID) signal for the selected NAV/DME or ADF receiver.

MENUS AND DISPLAYS Embraer Legacy 450/500 and Praetor 500/600 PFD Menu Pro Line Fusion® Avionics

EVS CONTROL The EVS CONTROL selection closes the PFD Task Menu and opens the EVS Format dialog box when selected. Refer to the ﬁgure on page 5-77. The EVS CONTROL selection is available when the optional EVS is installed.

HSI FORMAT The HSI FORMAT selection closes the PFD Task Menu and opens the HSI Format dialog box when selected. Refer to the ﬁgure on page 5-78.

COM/NAV SRC The COM/NAV SRC selection closes the PFD Task Menu and opens the COM/NAV SRC dialog box when selected. Refer to the ﬁgure on page 5-80.

BRG SRC The BRG SRC dialog allows the pilots to turn on or off two bearing pointers and select what navigation source drives each pointer. Bearing pointer 1 (BRG1) is driven by the left side sources (xxx1), bearing pointer 2 (BRG2) is driven by the right side sources (xxx2). Refer to the ﬁgure on page 5-81.

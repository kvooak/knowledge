MENUS AND DISPLAYS Embraer Legacy 450/500 and Praetor 500/600 Tuning – Edit HF Preset Frequencies Pro Line Fusion® Avionics

The Edit HF Preset Frequencies submenu provides controls to manage and edit user preset HF frequencies. The Edit HF Preset Frequencies submenu is accessed by pushing the EDIT pushbutton on the HF Control menu.

SELECTIONS:

Select HF Fre- quency

The Select HF Frequency list displays the currently stored HF preset frequencies. The user can select frequencies for editing and scroll the list with the inner knob on the DSK.

Set Mode The Set Mode window provides mode control for the HF preset frequencies. The user can set the mode of the presets by positioning the cursor on the Set Mode window and using the inner knob of the DSK.

Set Emissions The Set Emissions window provides emissions control for Simplex and Duplex HF preset frequencies. The user can set the emissions level of the presets by positioning the cursor and using the inner knob of the DSK. If the mode of the selected preset is Emergency or Maritime, the Set Emissions window is not shown.

Set Frequency The Set Frequency window provides tuning control for the HF preset frequencies. Select the frequency in the window to edit it.

OVERVIEW – SYSTEM COMPONENTS Embraer Legacy 450/500 and Praetor 500/600 System Description Pro Line Fusion® Avionics

Multifunction Keyboard Panel (MKP)

The Multifunction Keyboard Panel (MKP) provides an alphanumeric keyboard with a scratchpad display, and specialized display control keys. Two MKP are installed, one for the pilot side and one for the copilot side.

Multiscan™ Weather Radar System Re- ceiver/Trans- mitter Antenna (RTA)

The Multiscan™Weather Radar System Receiver/Transmitter Antenna (RTA) is located in the radome. The RTA is made up of an antenna, antenna tilt and scan motors, a receiver/transmitter, and signal processing circuits. RTA generated precipitation related weather and turbulence targets, as well as system operation and fault messages, can be shown on any of the ﬂight displays.

Navigation (NAV) Receiver The Navigation (NAV) receiver provides VOR/Localizer (LOC), glideslope, Automatic Direction Finder (ADF), and marker beacon outputs that can be shown on the ﬂight displays and for use by the navigation and hazard avoidance subsystems. The NAV receiver also receives identiﬁcation and voice audio. Two NAV receivers are installed.

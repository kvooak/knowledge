MENUS AND DISPLAYS Embraer Legacy 450/500 and Praetor 500/600 ADI – Lateral/Vertical Deviation Pro Line Fusion® Avionics

SELECTIONS:

Vertical Deviation Scale

The vertical deviation scale shows on the ADI when Localizer (LOC) is the active or preselect NAV source and backcourse logic is not active, or when FMS Vertical Navigation (VNAV) is active.

Vertical Deviation Pointer

The vertical deviation pointer displays Glideslope (GS) deviation information when LOC is the active NAV source. The GS vertical deviation pointer is a green diamond.

VNAV Deviation Pointer

The VNAV deviation pointer displays vertical deviation information when FMS is active and VNAV is commanded. The VNAV deviation pointer is a magenta waypoint symbol.

Lateral Deviation Scale

The lateral deviation scale shows on the ADI when Localizer (LOC) is the active or preselect NAV source.

Lateral Deviation Pointer

The lateral deviation pointer displays non-FMS lateral deviation information when FMS is not the active NAV source. The lateral deviation pointer is a green diamond.

FMS Lateral Devia- tion Pointer

The FMS lateral deviation pointer displays FMS lateral deviation information when FMS is the active NAV source and the lateral deviation scale shows. The FMS lateral deviation pointer is a magenta waypoint symbol.

MENUS AND DISPLAYS Embraer Legacy 450/500 and Praetor 500/600 VSD – Terrain Warning Pro Line Fusion® Avionics

• TAWS TEST shows in white when the TAWS is placed into test mode. When initially activated, TAWS TEST ﬂashes for 5 seconds. The TAWS TEST message and test pattern is removed when TAWS is taken out of test mode. The display returns to its previous conﬁguration when the TAWS test times out.

• TERRAIN FAIL is shown in yellow when the TAWS overlay display data is missing or invalid. When initially activated, the TERRAIN FAIL message ﬂashes for 5 seconds.

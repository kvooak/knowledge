Embraer Legacy 450/500 and Praetor 500/600 MESSAGES AND ANNUNCIATORS Pro Line Fusion® Avionics Visual Annunciations

#### HSI – CAUTION FLAGS

HSI: Lightning Detection Overlay Fault Annunciation

#### . A yellow LX FAIL annunciation message is shown in the overlay fault above the aircraft symbol on the HSI when the Lightning Detection System (LDS) is reporting itself failed. The message ﬂashes for 5 seconds and then changes to a steady-on state.

When the LX FAIL message is being shown, the LDS lightning bolt icons are removed from the display.

WX/LX FAIL shows if a WXR and LDS fault are to be shown simultaneously.

Embraer Legacy 450/500 and Praetor 500/600 CONTROLS AND INDICATORS Pro Line Fusion® Avionics Display Control Panel

ET Elapsed Time But- ton

The Elapsed Time (ET) button has three functions: The ﬁrst push of the ET button starts a timer, visible on the upper right portion of the HSI window in a normal conﬁguration. The second push of the ET button stops the timer, but leaves it visible on the HSI. Pushing the ET button a third time removes the ET display from the HSI.

TCAS Trafﬁc Collision Avoidance System Button

The Trafﬁc Collision Avoidance System (TCAS) button places trafﬁc overlay information on the HSI Map format.

WX Weather Radar Button

The Weather Radar (WX) button places weather overlay information on the HSI Map format.

BARO Barometric Setting Knob

The Barometric (BARO) setting knob allows the crew to change the barometric settings for the EFIS.

SET STD Set Standard Baro- metric Pressure Button

The SET STD button within the BARO knob automatically sets the Baro Set value to the Standard (STD) barometric pressure setting of 29.92 Inches of Mercury (inHg)/1013 Hectopascals (hPa). This pushbutton also swaps the active and preset barometric values when pushed. Changing between STD and barometric pressure toggles the preselect altitude format between XXXxx and FLXXX (for example, 19 000 and FL190).

Figure 5-148 XM GWX – Menu Bar

The XM GWX Menu Bar shows along the top of all XM GWX format pages. The XM GWX Menu Bar provides controls to navigate the XM GWX menus and to select a XM GWX display.

SELECTIONS:

Window Format Button

The window format button is a combo box to conﬁgure the DU formats.

OVLY OVLY is a menu button to show the XM GWX Overlay page and the Overlays menu.

Pan Control The pan control is a button with a hand symbol on it. The pan control is a single-state toggle (toggle on/toggle off), and times out (toggle off) after 2 seconds or cursor inactivity. If the cursor is moved again after the box has timed out, the box reappears. When the pan control is toggled on, moving the cursor instead pans the current GWX display.

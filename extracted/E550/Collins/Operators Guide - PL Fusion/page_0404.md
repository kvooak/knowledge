MESSAGES AND ANNUNCIATORS Embraer Legacy 450/500 and Praetor 500/600 Visual Annunciations Pro Line Fusion® Avionics

HSI: TCAS Resolution Advisory Fault

#### NO ATT RA (yellow)

A yellow NO ATT RA annunciation message is shown in the TCAS data block on the right side of the HSI if pitch data is not being shown on the ADI.

HSI: TCAS Fault Annunciation

#### . A yellow TCAS FAIL annunciation message is shown in the TCAS data block on the right side of the Map and HSI if the TCAS system is failed. A TCAS that is turned off is considered failed. TCAS OFF is displayed when the trafﬁc mode is standby, alt on, or alt off. Failures related exclusively to the trafﬁc display cause this message to be shown when the TCAS overlay is selected.

HSI: Weather Radar Overlay Fault Annunciation

#### . A yellow WX FAIL annunciation message is shown in the overlay fault ﬁeld above the aircraft symbol on the HSI when the Weather Radar (WXR) is not reporting mode information and the weather overlay is selected. The message ﬂashes for 5 seconds and then changes to a steady-on state.

Weather radar operating mode data is removed when this message is being shown.

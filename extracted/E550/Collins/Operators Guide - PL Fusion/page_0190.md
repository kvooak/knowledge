MENUS AND DISPLAYS Embraer Legacy 450/500 and Praetor 500/600 Radar Dialog Box (Multiscan Radar Installed) Pro Line Fusion® Avionics

SELECTIONS:

MODE MODE is a multi-state toggle to select the operating mode of the radar unit.

WXR WXR is a multi-state toggle to select the weather radar to ON or Standby (STBY).

TEST TEST is a selection used to check the display colors and system connections. After selecting test the radar test pattern will be sent from the radar to the displays.

TILT Use the inner knob on the CCP DSK or the tilt knob on the DCP to select the WXR tilt angle.

GAIN Use the outer knob on the CCP DSK to select the value of the WXR gain. Values are from +3 to −3.

TIP

Return the gain to NORM to make sure the radar is showing the proper returns.

Embraer Legacy 450/500 and Praetor 500/600 MENUS AND DISPLAYS Pro Line Fusion® Avionics Checklist

The Menu Bar is shown at the top of each Checklist page and provides the following controls:

SELECTIONS:

INDEX The INDEX drop down menu provides a method to return to the Normal index from any Checklist page.

QUEUE The Checklists show in prioritized order when the QUEUE button is selected.

FCN The FCN drop down menu provides the following selections:

• Reset Checklist – Resets all items in the currently shown Checklist to the Started state.

• Reset Index — Resets all checklists within an Index. Available only within an index.

• Reset All – Resets all Checklists to the Not Started state. This selection is selectable inside the Index only.

• Reset Override Items — Resets overidden checklists.

• Override Item – Sets the selected Checklist item to the Overridden state.

• Override Checklists – Sets the currently shown Checklist to the Overridden state. This selection is selectable inside a Checklist only.

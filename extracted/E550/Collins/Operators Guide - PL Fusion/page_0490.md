#### ROLL MODE

Roll mode is the basic mode for lateral axis. It is typically activated while FD is OFF and FD button is pushed, or when any vertical mode is activated, or AP is engaged, or any lateral mode is active and the respective button on the FCP is pushed. The Roll mode reference can be resynchronized to a new reference by pushing TCS buttons on the sidesticks, maneuvering the aircraft to a new roll angle and subsequently, releasing the TCS buttons.

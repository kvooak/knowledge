Embraer Legacy 450/500 and Praetor 500/600 MENUS AND DISPLAYS Pro Line Fusion® Avionics Tuning – Edit COM User Preset Frequencies

Set Frequency The SET FREQUENCY window provides tuning control for the user preset frequencies. Select the frequency in the window to edit it.

Deﬁne Name The DEFINE NAME window provides control to edit the shown name for the user preset frequencies. Select the Deﬁne Name window to edit its contents.

Return to VHF CTRL

Select the RETURN TO COMX CTRL pushbutton to close the Edit COM User Preset Frequencies window. The Edit COM User Frequencies window may also be closed by selecting the gray X pushbutton in the upper right corner.

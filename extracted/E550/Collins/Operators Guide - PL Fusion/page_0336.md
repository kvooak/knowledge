MENUS AND DISPLAYS Embraer Legacy 450/500 and Praetor 500/600 Chart – NOTAMs Window Pro Line Fusion® Avionics

The chart NOTAMs show one at a time, and a paging icon provides controls to advance to subsequent NOTAMs (if available). A Print button shows if a printer is installed. Selecting the Print button displays the Print NOTAMs dialog box.

Figure 5-143 Chart – Print NOTAMs Dialog Box

If two printers are installed, the Print NOTAMs dialog box includes a multi-state toggle to select the printer.

If no printers are installed, the Print button is removed from the Chart NOTAMs window.

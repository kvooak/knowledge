(LH/RH/CTR/ LWR) DU Display Control Knobs

LH DU, CTR DU, RH DU, and LWR DU knobs provide individual dimming control for each of the display units. Turning a dimming control clockwise increases the display brightness. Turning a control fully counterclockwise turns the associated display off. When a dimming control turns a display off, automatic reversion for the remaining displays is invoked so that dual PFDs and an EICAS are retained.

TUNE The TUNE inhibit switch is a three position switch with positions labeled NORM, L/R MKP ONLY, and COM1/2 121.5. The NORM position is the normal operating position when the radio tuning system is functioning properly. It allows for display tuning of the radios. Selection of the L/R MKP ONLY position inhibits tuning by the displays. Selection of the COM 1/2 121.5 position bypasses the system tuners and forces the COM1 or 2 VHF radio to directly tune to the 121.50 MHz emergency frequency.

ADS PFD Air Data Source Reversion

The ADS button provides a means to manually change the ADS source for the left and right PFD.

ATT HDG Attitude/Heading Source Control Buttons

The ATT HDG button provides a means to manually change the attitude and heading source for the PFD. The left PFD normally uses IRS1, and the right normally uses IRS2. Pushing the button cycles between IRS1 and IRS2.

Embraer Legacy 450/500 and Praetor 500/600 MENUS AND DISPLAYS Pro Line Fusion® Avionics Database Status Format

The Database Status format displays information about the installed periodic databases and tables. Periodic databases are databases that are updated on a speciﬁed regular schedule. Tables are any loadable ﬁles that are not application software or periodic databases.

The Database Status format displays the database name, status or part number, and begin and end dates of effectivity. Paging control is provided to show more than 20 databases.

Embraer Legacy 450/500 and Praetor 500/600 MENUS AND DISPLAYS Pro Line Fusion® Avionics Introduction

## MENUS AND DISPLAYS

### INTRODUCTION

This chapter describes the menus and displays that are used to operate the features that are available with Pro Line Fusion Avionics System installed in the Embraer aircraft. Each of the menus and displays that are or can be shown on the ﬂight displays are described in this chapter. Basic use of theCursor Control Panel (CCP), Display Control Panel (DCP), and Multifunction Keyboard Panel (MKP) is covered in the DISPLAY OPERATION and CONTROLS AND INDICATORS chapters. Operation of the Automatic Flight Control System (AFCS) is covered in the AFCS OPERATION chapter.

### DISPLAY UNIT The Display Unit (DU) provides a graphical representation of ﬂight information to the crew in the form of a PFD.

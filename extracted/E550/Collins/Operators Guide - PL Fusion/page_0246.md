MENUS AND DISPLAYS Embraer Legacy 450/500 and Praetor 500/600 Tuning – Edit ADF Preset Frequencies Pro Line Fusion® Avionics

### TUNING – EDIT ADF PRESET FREQUENCIES

Figure 5-85 Tuning – Edit ADF Preset Frequencies

The Edit ADF Preset Frequencies submenu provides controls to manage and edit user preset ADF frequencies. The Edit ADF Preset Frequencies submenu is accessed by pushing the EDIT pushbutton on the ADF Control menu.

SELECTIONS:

Select ADF Fre- quency

The Select ADF Frequency list displays the currently stored ADF preset frequencies. The user can select frequencies for editing and scroll the list with the inner knob on the DSK.

Set Frequency The Set Frequency window provides tuning control for the ADF preset frequencies. Select the frequency in the window to edit it.

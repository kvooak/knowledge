MENUS AND DISPLAYS Embraer Legacy 450/500 and Praetor 500/600 Synoptic – Bleed Air/Anti-Ice Page Pro Line Fusion® Avionics

Figure 5-59 Synoptic – Anti-Ice Page

The ANTI-ICE Synoptic page graphically represents the Bleed Air/Anti-Ice system. The ANTI-ICE Synoptic page is accessed through the Synoptic drop down menu accessed with the SYS QAK.

SELECTIONS:

Wing Anti-Ice Wing anti-ice temperature is depicted with a line section on the leading edge of each wing depiction. The color of each line section indicates the following status:

• Green line – on.

• White line – off.

• Yellow line – undetermined.

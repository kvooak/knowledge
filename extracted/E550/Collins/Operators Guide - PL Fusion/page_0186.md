MENUS AND DISPLAYS Embraer Legacy 450/500 and Praetor 500/600 Vspeeds / LFE Dialog Box Pro Line Fusion® Avionics

VAC VAC – Approach climb speed. This reference is shown on the ADI airspeed scales as a magenta horizontal line with an magenta AC legend.

VFS VFS – Final Segment climb speed. This reference is shown on the ADI airspeed scales as a green horizontal line with a green FS legend, during both takeoff and approach.

OAT The OAT selection allows the pilot to manually enter the Outside Air Temperature (OAT).

SEND The SEND button commands all manually-entered values to be sent to the FADEC and posted to the PFD.

LFE The LFE selection allows the pilots to enter the Landing Field Elevation (LFE) either manually or use the FMS sourced elevation.

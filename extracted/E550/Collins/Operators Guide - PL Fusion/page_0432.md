DISPLAY OPERATION Embraer Legacy 450/500 and Praetor 500/600 GUI Elements Pro Line Fusion® Avionics

• A single-state toggle control is represented by a check box, and is used for any situation where a function has two possible control states, ON or OFF. Position the cursor over the DU single-state toggle and select it to turn the associated function ON or OFF.

• A multi-state toggle is represented by a set of radio buttons, and is used for any function where there is more than one control state and only one of the states can be active at a time. Position the cursor over an option on the DU multi-state toggle and select it to enable the associated function.

• Enumerated data entry is used for any function where a value must be entered and the selection is from a limited set of predeﬁned values. Each enumerated data entry element has limits and increments deﬁned by its GUI. Position the cursor over the DU enumerated data entry and use the DSK to change the value.

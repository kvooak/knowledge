CONTROLS AND INDICATORS Embraer Legacy 450/500 and Praetor 500/600 Multifunction Keyboard Panel Pro Line Fusion® Avionics

Alphanumeric Key- pad

The alphanumeric keypad includes keys for each letter of the alphabet and each numerical digit, in addition to a Space (SP), a slash (/), a decimal point (.), a plus/minus toggle (+/−), and an ENTER button.

CNCL Cancel Button

The CNCL button is used to cancel an in-process ﬂight plan edit.

FMS FMS Quick Access Key

The FMS QAK displays a FMS format on the on-side MFW of the bottom display unit (DU4). The on-side cursor is positioned on the FMS format. Multiple pushes of the FMS QAK cycles through the FMS formats.

ROUTE Route Quick Ac- cess Key

Selecting the ROUTE QAK shows the ROUTE window format on the MFW.

MSG Message Quick Access Key

Pushing the MSG QAK shows or removes the FMS MESSAGES dialog box on the MFW.

DEP/ARR Departures/Arrivals Quick Access Key

While airborne, selecting the DEP/ARR QAK shows the ARRIVALS dialog box on a multifunction window. When on the ground, selecting the DEP/ARR button shows the DEPARTURES dialog box on a multifunction window.

Ɖ→ Direct-To Quick Access Key

Pushing the (Ɖ→) QAK brings up a Direct-To dialog box which is used to edit the ﬂight plan.

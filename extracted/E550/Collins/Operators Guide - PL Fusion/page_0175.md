### PFD FORMAT DIALOG BOX

Figure 5-31 PFD Format Dialog Box

The PFD Format dialog box provides controls to adjust the settings of the on-side PFD. The PFD Format dialog box shows by selecting PFD FORMAT from the PFD Task Menu.

SELECTIONS:

PFD PFD section is a multi-state toggle to select a FULL or HALF screen PFD format for the on-side DU.

XTRACKBALL XTRACKBALL is a multi-state toggle to inhibit the inputs to all displays from the cross-side CCP trackball when selected to OFF.

FPV CAGE FPV CAGE is a multi-state toggle to cage and uncage the FPV.

Embraer Legacy 450/500 and Praetor 500/600 MENUS AND DISPLAYS Pro Line Fusion® Avionics Chart – Airport Menu Windows

deﬁned by the Airport menu type. Selecting the chart shortcut submenu buttons for all submenu types except for Any Chart displays a Chart Submenu for the associated chart submenu type. Selecting the Any Chart shortcut submenu button displays the Chart Type menu.

SELECTIONS:

Airport The Airport submenu line displays the available airport charts at the associated airport.

Departure The Departure submenu line displays the available departure charts at the associated airport.

Arrival The Arrival submenu line displays the available arrival charts at the associated airport.

Approach The Approach submenu line displays the available approach charts at the associated airport.

Any Chart The Any Chart submenu line displays all of the available charts at the associated airport. Selecting the chart shortcut submenu button for Any Chart displays the Chart Type menu.

Chart NOTAMs Chart NOTAMs is a submenu access button to show the NOTAMs window for the associated airport. If there are no NOTAMs for the selected airport, the NOTAMs button is disabled.

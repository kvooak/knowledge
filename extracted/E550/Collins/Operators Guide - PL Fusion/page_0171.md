Embraer Legacy 450/500 and Praetor 500/600 MENUS AND DISPLAYS Pro Line Fusion® Avionics PFD Menu

Figure 5-28 PFD Menu (with optional HUD)

The PFD Task Menu provides controls to adjust the settings for the on-screen ﬂight information displays. The PFD Task Menu is displayed when the cursor is placed anywhere on the HSI and either the CCP MENU button or the CCP side select buttons are pushed. The PFD Task Menu is removed from the display 3 seconds after the cursor is moved off the menu.

SELECTIONS:

PFD FORMAT The PFD FORMAT selection closes the PFD Task Menu and opens the PFD Format dialog box when selected. Refer to the ﬁgure on page 5-73.

HUD FORMAT The HUD FORMAT selection closes the PFD Task Menu and opens the HUD Format dialog box when selected. Refer to the ﬁgure on page 5-75. The HUD FORMAT selection is available when the optional HGS is installed.

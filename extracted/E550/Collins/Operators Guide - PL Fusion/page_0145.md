Embraer Legacy 450/500 and Praetor 500/600 MENUS AND DISPLAYS Pro Line Fusion® Avionics ADI – Vertical Speed

• The readout, icon, and bug are shown in magenta when the FMS is in control. Clearing the VS mode or selecting another vertical mode removes the selected VS bug and associated display in the AFCS mode ﬁeld.

VSR Bug The FMS Vertical Speed Required (VSR) advisory bug is a magenta open circle that is shown on the VS scale. The FMS VSR advisory bug indicates the minimum average VS required to climb (or descend) from the present position to the next climb (or descent) altitude constraint. The speciﬁc conditions under which the FMS VSR advisory bug is shown are controlled by the FMS. The FMS must be the active NAV source for the bug to be shown.

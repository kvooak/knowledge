### FLIGHT DIRECTOR SELECT

With the AP not engaged, the pilot can manually ﬂy the aircraft in response to AFCS commands that are shown on the ADI format by the Flight Director. When the AP is engaged, the AP maneuvers the aircraft in response to AFCS commands and the pilot monitors the ﬂight path by observing the Flight Path Vector (FPV). The FD comes into view on the displays when a lateral or vertical mode is selected or the autopilot is engaged. AFCS mode messages are shown on the displays when a FD is selected on either side.

MENUS AND DISPLAYS Embraer Legacy 450/500 and Praetor 500/600 Tuning – Edit NAV Preset Frequencies Pro Line Fusion® Avionics

### TUNING – EDIT NAV PRESET FREQUENCIES

Figure 5-81 Tuning – Edit NAV Preset Frequencies

The Edit NAV Preset Frequencies submenu provides controls to manage and edit user preset NAV frequencies. The Edit NAV Preset Frequencies submenu is accessed by pushing the EDIT pushbutton on the NAV Control menu.

SELECTIONS:

Select NAV Fre- quency

The Select NAV Frequency list displays the currently stored NAV preset frequencies. The user can select frequencies for editing and scroll the list with the inner knob on the DSK.

Set Frequency The Set Frequency window provides tuning control for the NAV preset frequencies. Select the frequency in the window to edit it.

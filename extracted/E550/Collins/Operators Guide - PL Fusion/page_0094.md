CONTROLS AND INDICATORS Embraer Legacy 450/500 and Praetor 500/600 Reversion Switch Panel Pro Line Fusion® Avionics

EICAS Engine Indication and Crew Alert System

The Engine Indication and Crew Alert System (EICAS) reversion switch places the EIS and CAS windows on either side of the HSI when using full-screen PFD, and a 1/4 MFW EICAS display on the inside, upper portion of the PFD when using a half-screen PFD. When one screen is inoperative the manual reversion places them in various places, depending on which screens are available. Refer to the Display Reversion sections of the Display Operations chapter of this operator’s guide.

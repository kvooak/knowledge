CONTROLS AND INDICATORS Embraer Legacy 450/500 and Praetor 500/600 Flight Control Panel Pro Line Fusion® Avionics

PUSH DIRECT Push Direct button

The PUSH DIRECT button gives the user a direct-to course to point the Course Deviation Indicator (CDI) directly at the station. When FMS is the primary NAVAID use the PUSH DIRECT button to point the CDI directly to the next waypoint in the ﬂight plan.

AT Autothrottle En- gagement Button

The AT engages and disengages the autothrottle.

FMS MAN 2 Position Rotary Pushbutton

Controls speed control selection between FMS and Manual.

SPD Speed Reference Control Knob and Readout

The SPD knob selects the desired airspeed, which is displayed on the PFD with an (M) indication whenever Mach target is selected. The SPD button toggle the speed unit between IAS and Mach.

PUSH IAS/MACH Speed IAS/Mach Switch

The PUSH IAS/MACH button within the SPD knob is used to alternatively select either IAS or Mach for the airspeed tape and speed bug reference. The IAS/MACH button only changes the speed reference not the speed tape. The corresponding speed reference value is in knots or Mach. This function overrides automatic IAS/Mach selection.

LNAV Lateral Navigation Mode Select Button

The LNAV button selects and deselects the lateral navigation mode. When in lateral navigation mode, the system generates commands to capture and track the selected lateral navigation source.

Embraer Legacy 450/500 and Praetor 500/600 MENUS AND DISPLAYS Pro Line Fusion® Avionics Tuning – HF Control

Deﬁne Name The Deﬁne Name window provides control to edit the shown name for the ADF preset frequencies. Select the Deﬁne Name window to edit its contents.

Return to ADF CTRL

Select the Return to ADF CTRL pushbutton to close the Edit ADF Preset Frequencies window. The Edit ADF Preset Frequencies window may also be closed by selecting the gray X pushbutton in the upper right corner.

### TUNING – HF CONTROL

Figure 5-86 Tuning – HF Control

The HF Control submenu provides advanced controls to modify HF settings. The HF Control menu is accessed by selecting a gray HF button on the Tuning format.

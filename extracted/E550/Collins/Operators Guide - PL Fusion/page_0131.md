Embraer Legacy 450/500 and Praetor 500/600 MENUS AND DISPLAYS Pro Line Fusion® Avionics ADI – Altitude

• BARO MIN readout is automatically shown when BARO is selected. An active BARO MIN displays to the bottom left of the ALT tape as XXXXX BARO, where XXXXX is the set BARO MIN value.

RA/BARO MIN Analog

The RA/BARO MIN analog reference is shown on the left side of the altitude scale at the value set for the RA/BARO MIN. The MIN analog reference is only shown when the RA/BARO MIN readout is in view.

• The RA analog reference is a cyan barber pole with black ﬁll, which extends up from the analog radio altitude display. The analog reference extends up toward the radio altitude 0 (zero) foot tic mark as the radio altitude decreases. At RA MIN alert, the RA MIN analog reference changes to yellow, ﬂashes for 5 seconds, and then remains steady.

• The BARO MIN analog reference is a cyan open triangle shaped bug that is shown on the ﬁne altitude scale at the selected BARO MIN altitude. The bug moves up the altitude scale as the barometric altitude decreases. At BARO MIN alert, the BARO MIN analog reference changes to yellow, ﬂashes for 5 seconds, and then remains steady.

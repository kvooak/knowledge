MENUS AND DISPLAYS Embraer Legacy 450/500 and Praetor 500/600 Map – Menu Bar Pro Line Fusion® Avionics

The Map sub-window section is a multi-state toggle. The overlay section is composed of single-state toggles to select the Terrain and Trafﬁc (TFC) overlays. The weather information section is a multi-state toggle. The GWX overlay section is composed of single-state toggles to show GWX images.

The age of each GWX product shows to the right. If the age is white, the image is fresh. If the age is yellow, the image is stale. If the image is not available, the age is replaced with dashes. The current UTC shows at the bottom of the menu.

GWX Altitude The GWX data entry ﬁeld shows when WINDS ALOFT, TURBULENCE, ICING, or SATELLITE are selected on the Map Overlay menu. The GWX ALT legend above the data entry ﬁeld changes to depict the selected GWX product. When the cursor is positioned on the GWX altitude data entry ﬁeld, use the outer knob of the DSK to change the altitude of the shown GWX overlay.

Select Map Item The Select Map Item dialog box is displayed when more than one highlights map symbol is selected.

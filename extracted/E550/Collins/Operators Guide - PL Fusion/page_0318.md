MENUS AND DISPLAYS Embraer Legacy 450/500 and Praetor 500/600 DU – Chart Format Pro Line Fusion® Avionics

N

NOTE

The use of Charts with the geographically referenced aircraft symbol does not relieve the pilot of responsibility for navigation.

Figure 5-131 Chart – Menu Bar

The Chart Menu Bar shows along the top of all Chart MFW formats. The Chart Menu Bar provides controls to navigate the Chart menus and show a chart.

SELECTIONS:

M The DU menu is accessed by selecting the M with the CCP. The menu is the same as pushing the MENU button on the CCP.

ORIG ORIG is a button to access the Origin Chart menu. If no Origin airport is speciﬁed in the FMS ﬂight plan, the ORIG button is disabled.

DEST DEST is a button to access the Destination Chart menu. If no Destination airport is speciﬁed in the FMS ﬂight plan, the Destination button is disabled.

ALTN ALTN is a button to access the Alternate Chart menu. If no Alternate airport is speciﬁed in the FMS ﬂight plan, the Alternate button is disabled.

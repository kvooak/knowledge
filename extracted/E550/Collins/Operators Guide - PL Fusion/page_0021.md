Embraer Legacy 450/500 and Praetor 500/600 INTRODUCTION Pro Line Fusion® Avionics

## INTRODUCTION

This guide describes the components and operation of the Pro Line Fusion® Avionics System for the Embraer Legacy 450/500 aircraft. The general operational information in this guide must be supplemented with information contained in the Aircraft Flight Manual for the Embraer Legacy 450/500 and the following major subsystem publications:

• Rockwell Collins Embraer Legacy 450/500 Flight Management System Operator’s Guide (CPN 523-0818863).

• Rockwell Collins RTA-4100 Series MultiScan™Automatic Weather Radar Operator’s Guide (CPN 523-0816797).

• Rockwell Collins ATN and FANS Harmonized CPDLC Application Operator’s Guide (CPN 523-0822822).

An overview of each of the major Rockwell Collins avionics subsystems that make up the Pro Line Fusion Avionics System follows this introduction. Information about the operation of the displays is contained in the appropriate operation chapters that follow the overview chapters. The Controls and Indicators chapter covers the control panels and display controls the pilot uses in operating the on-board systems. Information regarding messages, annunciations, and ﬂags that require action from the pilot, or that indicate a malfunction or failure of a system, is contained in the Messages and Annunciators chapter. Further information is found in the appendixes. These include:

• Appendix A: Exploded Menu Trees — Gives an exploded view of the PFD menu (full screen and compressed), Display Unit main menu, and FMS map main menus.

• Appendix B: TCAS Aurals — Gives a description of each of the TCAS callouts for trafﬁc separation.

• Appendix C: CPDLC — Gives a description of the CPDLC/ATN interface.

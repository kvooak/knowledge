CONTROLS AND INDICATORS Embraer Legacy 450/500 and Praetor 500/600 Cursor Control Panel Pro Line Fusion® Avionics

CALLOUTS:

DSK Double Stack Knob

The Double Stack Knob (DSK) is a general purpose control device for DU Graphic/User Interface (GUI) applications. The DSK is a dual concentric knob composed of a large, outer knob and a small, inner knob.

The DSK has speciﬁc functions based on the active DU format.

PUSH ENTER Enter Button

The PUSH ENTER button inside the DSK is used to select options for DU GUI applications.

The PUSH ENTER button has speciﬁc functions based on the active DU format.

MENU Menu Button

Pushing the MENU button when the cursor is in a Multifunction Window (MFW) displays the top-level drop down menu for that format. When the cursor is not in a MFW, pushing the MENU button displays a SELECTION INACTIVE message on the on-side HSI for two seconds.

XPDR IDENT Transponder Ident

Push the XPDR IDENT button to send the aircraft transponder information to Air Trafﬁc Control (ATC).

DSPL SEL Display Select But- tons

The DSPL SEL buttons show a cursor on the Display Units (DU). Each CCP has four DSPL SEL buttons, and the buttons operate the same on each side of the cockpit (pilot, copilot). The buttons are arranged in the order of screen they access.

MENUS AND DISPLAYS Embraer Legacy 450/500 and Praetor 500/600 PFD Format Dialog Box Pro Line Fusion® Avionics

PFD CVS PFD CVS is a multi-state toggle to select Combined Vision System (CVS) images for display on the PFD. The CVS button selects between OFF, Synthetic Vision System (SVS), Enhanced Vision System (EVS), and Enhanced Synthetic Vision System (ESVS).

Figure 5-32 PFD – Compressed Dialog Box

Figure 5-33 PFD – Compressed Dialog Box

When the PFD is compressed, the PFD Format dialog box is compressed as well. The PFD FULL/HALF and PFD CVS selections are removed.

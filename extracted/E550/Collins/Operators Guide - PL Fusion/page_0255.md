Embraer Legacy 450/500 and Praetor 500/600 MENUS AND DISPLAYS Pro Line Fusion® Avionics Tuning – Error Messages

Figure 5-90 Tuning – Display Tuning Inhibited

The Display Tuning Inhibited message is displayed when both the pilot and copilot side RSP (Reversion Select Panel) TUNE knobs have been placed in the X MKP ONLY positions. When the knobs are placed in this position, display of COM/NAV tuning is inhibited on the display units. The COM/NAV radios can still be tuned, but they must be tuned via the MKP.

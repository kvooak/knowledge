Embraer Legacy 450/500 and Praetor 500/600 MENUS AND DISPLAYS Pro Line Fusion® Avionics Tuning – NAV Control

DME Hold DME Hold is a single-state toggle to enable or disable DME hold. The selected DME frequency shows in the frequency window when DME hold is enabled.

MKR MKR is a multi-state toggle to select LO or HI marker beacon sensitivity.

NAV Tuning NAV Tuning is a multi-state toggle to select AUTO or MAN as the NAV tuning mode. AUTO is selected upon power-up, but is disabled when automatic NAV tuning is unavailable.

Preset Frequencies The lower control menu lists the preset frequencies, controlled by a scroll list. The preset frequencies shown is deﬁned by the selection in the FREQ multi-state toggle. Select a preset frequency to set it as the active frequency.

Edit Edit is a pushbutton to show the Edit NAV Preset Frequency dialog box.

Test Test is a pushbutton to initiate a NAV test.

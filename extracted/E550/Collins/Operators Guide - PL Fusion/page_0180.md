MENUS AND DISPLAYS Embraer Legacy 450/500 and Praetor 500/600 PFD – HSI Format Dialog Box Pro Line Fusion® Avionics

SELECTIONS:

CALIBRATE The CALIBRATE button is used to command a manual calibration of the EVS sensor. Calibration is normally performed automatically by the EVS camera during most phases of ﬂight, except when airborne below 1000 feet, or when on the ground with a ground speed > 40 knots.

HEAT HEAT controls the heater for the window in front of the EVS camera. AUTO is the standard setting. The OFF selection is not available during ﬂight. The OFF selection is intended for maintenance operations.

### PFD – HSI FORMAT DIALOG BOX

Figure 5-38 HSI Format Dialog Box

The HSI Format dialog box provides controls to select the overlays for the on-side HSI. The HSI Format dialog box shows by selecting HSI FORMAT on the PFD Task Menu.

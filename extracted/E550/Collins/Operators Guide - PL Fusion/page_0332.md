MENUS AND DISPLAYS Embraer Legacy 450/500 and Praetor 500/600 Chart – Split Format Window Pro Line Fusion® Avionics

The Split format displays the chart in the top half of the window and a subwindow in the bottom half. The operator can select the data shown in the bottom subwindow.

SELECTIONS:

Brieﬁng Brieﬁng is a selection in a multi-state toggle to show the brieﬁng data in the bottom subwindow.

Proﬁle Proﬁle is a selection in a multi-state toggle to show the approach proﬁle view in the bottom subwindow.

Minimums Minimums is a selection in a multi-state toggle to show the minimums data in the bottom subwindow.

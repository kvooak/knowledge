### PFD – BARO SET DIALOG BOX

Figure 5-47 PFD – Baro Set Compressed Dialog Box

Figure 5-48 PFD – Baro Set Dialog Box

The BARO Set dialog box provides controls to set the on-side Barometric (BARO) pressure setting, and to select the BARO units.

N

NOTE

When the HSI is compressed, the BARO Set dialog box is compressed as well. The compressed BARO Set dialog box retains all of the selections of the non-compressed BARO Set dialog box, but the blue menu bar and gray X at the top of the menu are removed.

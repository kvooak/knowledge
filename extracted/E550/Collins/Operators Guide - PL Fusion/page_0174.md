Figure 5-30 PFD Compressed Menu (with optional HUD)

When the HSI is compressed, the PFD Task Menu is compressed as well. The compressed menu is displayed when the cursor is placed anywhere on the compressed HSI and either the CCP MENU button or the CCP side select buttons are pressed. It can be closed by clicking anywhere off of the menu, for example in the cutout for the heading indication, or anywhere else on the display. The compressed PFD Task Menu is removed after 3 seconds if the cursor is moved off the menu.

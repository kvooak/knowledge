DISPLAY OPERATION Embraer Legacy 450/500 and Praetor 500/600 DU Conﬁguration and Reversion Pro Line Fusion® Avionics

• Memorized Conﬁguration Storage: Each of the nine memorized conﬁgurations may be conﬁgured by the user. However, the names of the nine conﬁgurations may not be changed. To store the present DU conﬁguration, push and hold the MEM key on the MKP. The on-side MKP scratchpad shows MEM STORE 4-9, and the MEM STORE information box shows on the on-side HSI.

• Push a number key (4-9) on the MKP to store the present DU conﬁguration. This removes the MEM STORE information box, and MEM # STORED shows on the MKP for 3 seconds, where the # character is the number of the stored conﬁguration. Push the MEM key without pushing a number to cancel.

• Memorized Conﬁguration Recall: To recall a memorized conﬁguration, push the MEM key on the MKP. The on-side MKP scratchpad shows MEM RECALL 1-9, and the MEM RECALL information box shows on the on-side HSI.

• Push a number key (1-9) on the MKP to recall a memorized conﬁguration. This reconﬁgures the DU panels to the memorized conﬁguration. It also removes the MEM RECALL information box, and shows MEM # RECALLED on the MKP for 3 seconds (the # character is the number of the recalled conﬁguration). Push the MEM key without pushing a number to cancel.

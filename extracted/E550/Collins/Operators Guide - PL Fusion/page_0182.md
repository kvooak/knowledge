### HSI – COM / NAV SOURCE DIALOG BOX

Figure 5-39 PFD – NAV Source Compressed Dialog Box

Figure 5-40 PFD – NAV Source Dialog Box

The COM/NAV SRC dialog box provides controls to select the active COM, NAV, and FMS sources for the on-side MFD.

N

NOTE

When the HSI is compressed, the COM/NAV SRC dialog box is compressed as well. The compressed COM/NAV SRC dialog box retains all of the selections of the non-compressed COM/NAV SRC dialog box, but the blue menu bar and gray X at the top of the menu are removed.

SELECTIONS:

COM COM is a multi-state toggle to select the on-side COM source. The COM radio that is tuned when the MKP COM button is pressed after entering a COM radio frequency on the MKP.

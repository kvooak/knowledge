### CHART – AIRPORT MENU WINDOWS

Figure 5-132 Chart – Menu Window (Origin)

The Origin Airport menu provides controls to select charts corresponding to the Origin airport in the FMS ﬂight plan. If no Origin airport is speciﬁed in the FMS ﬂight plan or there is no ﬂight plan available, the Origin Airport menu is not available.

Figure 5-133 Chart – Menu Window (Destination)

The Destination Airport menu provides controls to select charts corresponding to the Destination airport in the FMS ﬂight plan. If no Destination airport is speciﬁed in the FMS ﬂight plan or there is no ﬂight plan available, the Destination Airport menu is not available.

MENUS AND DISPLAYS Embraer Legacy 450/500 and Praetor 500/600 Tuning – Edit COM User Preset Frequencies Pro Line Fusion® Avionics

### TUNING – EDIT COM USER PRESET FREQUENCIES

Figure 5-79 Tuning – Edit COM User Preset Frequencies

The Edit COM User Preset Frequencies submenu provides controls to manage and edit user preset VHF frequencies. The Edit COM User Preset Frequencies submenu is accessed by pushing the EDIT pushbutton on the COM Control menu when USER is selected on the FREQ control.

SELECTIONS:

Select User Fre- quency

The SELECT USER FREQUENCY list displays the currently stored user preset frequencies. The user can select frequencies for editing and scroll the list with the inner knob on the DSK.

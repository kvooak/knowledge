Embraer Legacy 450/500 and Praetor 500/600 MENUS AND DISPLAYS Pro Line Fusion® Avionics Radar Dialog Box (Multiscan Radar Installed)

Figure 5-52 PFD Multiscan Dialog Box

When a Multi-Scan™Weather Radar is installed this dialog box opens when the RADAR menu is selected on the PFD Task Menu. Refer to the Rockwell Collins RTA-4100 Series MultiScan™Automatic Weather Radar Operator’s Guide (CPN 523-0816797) for more information.

SELECTIONS:

MODE MODE is a multi-state toggle to select the operating mode of the radar unit.

WXR WXR is a multi-state toggle to select the weather radar to ON or Standby (STBY).

TEST TEST is a selection used to check the display colors and system connections. After selecting test the radar test pattern will be sent from the radar to the displays.

CTRL When set to AUTO, the radar adjusts tilt to follow the pitch of the aircraft automatically and calculated returns based on several different databases. When set to MAN the tilt angle is set by the pilots.

MENUS AND DISPLAYS Embraer Legacy 450/500 and Praetor 500/600 Chart – Airport Menu Windows Pro Line Fusion® Avionics

Figure 5-134 Chart – Menu Window (Alternate)

The Alternate Airport menu provides controls to select charts corresponding to the Alternate airport in the FMS ﬂight plan. If no Alternate airport is speciﬁed in the FMS ﬂight plan or there is no ﬂight plan available, the Alternate Airport menu is not available.

Figure 5-135 Chart – Menu Window (Other)

The Other Airport menu provides controls to select charts corresponding to the Other airport entered by the operator. If no Other airport has been entered by the operator, the Other button on the Chart Menu Bar brings up the on-screen Data Entry window.

Each Airport menu displays the ICAO identiﬁer and long name of the associated airport, and ﬁve chart shortcut submenu lines. The shortcut submenu lines consist of a readout of available charts and a shortcut submenu access button.

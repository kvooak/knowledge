Embraer Legacy 450/500 and Praetor 500/600 CONTROLS AND INDICATORS Pro Line Fusion® Avionics Flight Control Panel

### FLIGHT CONTROL PANEL

The Flight Control Panel (FCP) is installed at the center of the glareshield. The FCP contains ﬂight guidance mode select pushbuttons, the AP and AT engage and disengage pushbuttons, and FD select pushbuttons. An altitude select knob, a speed command knob, a heading set knob, a ﬂight path angle set knob, two course set knobs, and a vertical speed/pitch control thumbwheel are also provided. The ﬂight guidance mode buttons are push-on/push-off logic. A ﬂight guidance mode may be removed either by deselecting it or by selecting a different mode that is non-compatible.

CALLOUTS:

FD FD Select Button

The Flight Director button engages and disengages the ﬂight director on both PFD, showing and removing the FD from both displays simultaneously when applicable. The FD commands can not be removed when the AP is engaged.

CRS Course Select Con- trol Knob

The Course knob adjust the selected course on the left display. The button (PUSH DIRECT) synchronize the selected course on the left display to VOR Bearing when the navigation source is tuned to VOR.

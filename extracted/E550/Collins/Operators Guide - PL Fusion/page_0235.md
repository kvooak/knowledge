Embraer Legacy 450/500 and Praetor 500/600 MENUS AND DISPLAYS Pro Line Fusion® Avionics Tuning – COM Control

Mode Mode is a multi-state toggle to select the VHF mode between voice and data modes on the VHF3 radio (and optionally VHF2). If data is selected, the other VHF radio is automatically switched to voice mode. Selecting the data mode also disables the voice VHF controls on this menu.

FREQ FREQ is a multi-state toggle to select the preset frequency list. The selections is User, origin ICAO, destination ICAO, and alternate ICAO. The origin, destination, and alternate ICAO are sourced from the active ﬂight plan.

Preset Frequencies The lower control menu lists the preset frequencies, controlled by a scroll list. The preset frequencies shown is deﬁned by the selection in the FREQ multi-state toggle. Select a preset frequency to set it as the active frequency.

Edit Push the Edit button to show the Edit VHF User Frequency dialog box. The Edit control is only shown when the FREQ selection is USER.

Test Test is a pushbutton to initiate a VHF test.

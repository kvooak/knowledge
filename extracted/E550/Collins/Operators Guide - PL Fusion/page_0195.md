Embraer Legacy 450/500 and Praetor 500/600 MENUS AND DISPLAYS Pro Line Fusion® Avionics EICAS Format

The Engine Indication and Crew Alert System (EICAS) format provides essential engine and aircraft information. In normal operation, the EICAS format shows in the DU 2 Upper position, and cannot be moved.

When reversionary mode is selected with the EICAS button on the RSP, the EICAS display is moved to the bottom of the PFD, with the Engine Indicating System (EIS) showing left of the HSI, and the Crew Alerting System (CAS) showing right of the HSI. Each side may place the EIS information on the on-side PFD. The following information is provided:

• Engine parameters

• Fuel quantities

• Aircraft conﬁguration

• Trim information

• Crew Alerting System (CAS) message list.

SELECTIONS:

N1 The Engine Fan/Low Pressure Compressor (N1) display consists of analog scales, digital readouts, and command bugs for the N1 values, as well as V bugs and digital readouts for the target N1 rating.

• Analog scale – The N1 analog scale shows as a white arc. A green pointer displays the current N1 value.

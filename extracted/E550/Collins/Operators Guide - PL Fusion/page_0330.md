MENUS AND DISPLAYS Embraer Legacy 450/500 and Praetor 500/600 Chart – Chart Browse List Pro Line Fusion® Avionics

#### The Chart Browse list provides controls to quickly select other charts for the same airport as the currently shown chart. The Chart Browse list shows when the operator selects the Chart Status Line of the Chart Menu Bar. The available charts are listed in the same way as on the Chart Submenu.

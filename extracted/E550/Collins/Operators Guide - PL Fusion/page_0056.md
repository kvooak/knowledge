OVERVIEW – SYSTEM COMPONENTS Embraer Legacy 450/500 and Praetor 500/600 System Description Pro Line Fusion® Avionics

Adaptive Flight Display (AFD) The Adaptive Flight Display (AFD) units are color, Liquid Crystal Display (LCD) panels that provide display and control functions for the ﬂight instruments, ﬂight guidance, engine instruments, crew alerting system, navigation, hazard avoidance, and communications systems. Four AFDs are installed.

Cursor Control Panel (CCP) The Cursor Control Panel (CCP) is used to select the cursor on the displays by using the display select buttons, and to move the cursor using the trackball. Each CCP provides display select buttons, Menu button, a Double Stack Knob (DSK), transponder IDENT button, and Push To Talk (PTT) button. Two CCP are installed.

Data Concen- tration System (DCS)

The Data Concentration System (DCS) collects and organizes data from the onboard systems that interface with the Rockwell Collins avionics system. The DCS generates visual and aural information for use by the ﬂight displays and the ﬂight deck audio system. The DCS controls the setting and clearing of the ﬂight deck Master Warning (MW) and Master Caution (MC) lamps based on the setting and clearing of warning and caution Crew Alerting System (CAS) messages. The DCS consists of one Data Concentrator Module Cabinet (DMC) with two channels.

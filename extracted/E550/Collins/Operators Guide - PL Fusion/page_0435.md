Embraer Legacy 450/500 and Praetor 500/600 DISPLAY OPERATION Pro Line Fusion® Avionics GUI Elements

• The DU combo box is a combination of a push button with menu selection properties and menu. Position the cursor over the combo box and select it to open the submenu.

• The DU menu bar contains functionality that is speciﬁc to the format in view, and contains a format selection button. The menu bar is visible at the top of all selectable window formats. Position the cursor over a pushbutton on the menu bar and push the trigger to select the function or open the combo box.

• The Menu (M) pushbutton on the menu bar is used to show the MFW conﬁguration menu.

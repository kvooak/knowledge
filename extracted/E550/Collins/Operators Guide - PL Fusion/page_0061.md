Embraer Legacy 450/500 and Praetor 500/600 OVERVIEW – SYSTEM COMPONENTS Pro Line Fusion® Avionics System Description

Integrated Pro- cessing System (IPS)

The Integrated Processing System (IPS) consists of two IPC is a part of the on-board wiring and physically houses some avionics Line Replaceable Unit (LRU) modules. Each IPC contains its own Power Environment Module (PEM), Digital Switching Module (DSM), and several CCM. The IPC units host the following software applications on the CCMs: FMS 1, FMS 2, Terrain Awareness Warning System (TAWS), SVS, Integrated Flight Information System (IFIS), optional second IFIS, Datalink, emulated Control Display Unit (eCDU), and the OMS. IPC Picture: Legacy 450/500 only has a Left and Right IPC. The left IPC contains PEM, DSM, CCM1, CCM2, CCM3. The right IPC contains PEM, DSM, CCM1, CCM2, CCM3, TAWS Processing Module (TPM), SVM.

Lightning De- tection System (LDS) Proces- sor

The non-Rockwell Collins Lightning Detection System (LDS) processor calculates the azimuth and range of the electrical discharges within approximately 200 NM of the present position. The LDS processor generates lightning symbology as well as operating and fault messages that can be shown on the ﬂight displays.

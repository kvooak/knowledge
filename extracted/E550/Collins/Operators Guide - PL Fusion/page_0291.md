Embraer Legacy 450/500 and Praetor 500/600 MENUS AND DISPLAYS Pro Line Fusion® Avionics Selected Trafﬁc Data Block

The selected trafﬁc data block is present on a trafﬁc display whenever there is a selected trafﬁc on the display. The selected trafﬁc data block contains four data readouts for the selected trafﬁc:

• Flight ID (up to eight characters)

• Type

• Ground speed (in knots)

• Range (in nautical miles).

Flight ID is the alphanumeric aircraft identiﬁcation that is entered on a ﬂied ﬂight plan. For non-commercial operators, the ﬂight ID matches the registration number of the aircraft.
